Когда цикл while проверяет свое условие: до или после того, как он выполнит итерацию?
```
до
```
Сколько раз сообщение «Python awesome!» будет напечатано в приведенном ниже фрагменте кода?
count = 10
while count < 1:
print(‘Python awesome!’)
```
0
```
Сколько раз сообщение «Python awesome!» будет напечатано в приведенном ниже фрагменте кода?
count = 1
while count < 10:
print(‘Python awesome!’)
count += 1
```
9
```
